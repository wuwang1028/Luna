# Luna - 信息收集工具集成平台

Luna是一个基于Python开发的CLI工具，旨在整合和统一调用多个常用的信息收集工具，提供一站式的安全测试解决方案。

## 项目状态

🚧 **开发中** - 当前处于需求分析和设计阶段

## 项目目标

将以下7个信息收集工具进行整合，提供统一的调用接口和结果管理：

1. **OneForAll** - 子域名收集工具（Python）
2. **httpx** - HTTP探测工具（Go）
3. **ffuf** - Web模糊测试工具（Go）
4. **dirsearch** - 目录爆破工具（Python）
5. **fscan** - 内网综合扫描工具（Go）
6. **TXPortMap** - 端口扫描工具（Go）
7. **puzzle** - 一体化信息收集工具（Go）
